Javascript relationship with HTML and CSS

Javascript, HTML and CSS can be described as building a house. HTML can be described as the structure or outline of a house, CSS is the decoration of the house and Javascript is compared to the utilities of a house, having interactivity with the house.

Control flow and loops

Control flow is the process of reading code that is written in order, however the order can be changed with the use of if, else or else if. This also applies to loops like for loops or while loops. Loops are used to repeat a function until a certain condition is met.

DOM

The DOM can be described as a detailed map of the elements on your webpage. When you want to make changes to your website, you use javascript to modify the DOM.

Arrays and Objects

Arrays and Objects are used to store different types of data. Objects primary use is for storing data within keys, when used it labels an identifier and to call back to it. Arrays are similar and easier to explain, when data is stored in an array, it uses an index number. In coding, the index number always starts from 0 and counts up.

Functions

Functions are blocks of code used to perform a specific task. This avoids the necessity to rewrite code and it also requires the process of taking an input then outputting a result.
